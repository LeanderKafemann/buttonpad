Notes
-----

The package exposes the following public classes and functions:

- ``ButtonPad`` — the main window/layout class
- ``BPButton``, ``BPLabel``, ``BPTextBox``, ``BPImage`` — element wrappers
- ``alert``, ``confirm``, ``prompt``, ``password`` — thin wrappers around PyMsgBox with refocus behavior
